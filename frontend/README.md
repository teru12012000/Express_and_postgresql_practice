# frontend

## 概要

メンバー登録と閲覧ができるサイト

## 機能

- 閲覧
  - useEffect を使用してページロード時にデータを取得する
  - 表示の仕方は表で表している
  - 項目は ID,名前,年齢,趣味としている
  - メールとパスワードは見せないようにしている
- 登録
  - 追加ボタンクリック時にサーバー側の POST メソッドを起動させる
  - 正常に追加できたら成功のページへルーティングして閲覧のページへ誘導する
  - 何らかのエラーの場合にはアラートで知らせる

## 使用した技術

- Next.js
- TypeScript
- Bootstrap
- Materual-ui
